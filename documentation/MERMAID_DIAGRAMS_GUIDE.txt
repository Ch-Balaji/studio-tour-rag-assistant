================================================================================
HOW TO USE IN POWERPOINT
================================================================================

Method 1: Insert PNG (Simple)
------------------------------
1. Insert > Pictures
2. Select architecture_mermaid_enhanced.png
3. Resize as needed (maintains quality)
4. Recommended for most presentations

Method 2: Insert SVG (Best Quality)
------------------------------------
1. Insert > Pictures
2. Select architecture_mermaid_enhanced.svg
3. Can be scaled infinitely without quality loss
4. Can be ungrouped and edited in PowerPoint (advanced)
5. Recommended for professional presentations

Method 3: Edit and Re-render
-----------------------------
1. Open architecture_enhanced.mmd in text editor
2. Make changes to the Mermaid code
3. Visit https://mermaid.live/
4. Paste the code
5. Export as PNG or SVG
6. Use in your presentation

EDITING MERMAID DIAGRAMS
================================================================================

The .mmd files contain the source code for the diagrams. You can edit them
to customize the architecture diagram.

Online Editor:
--------------
1. Go to https://mermaid.live/
2. Copy the contents of any .mmd file
3. Paste into the editor
4. Edit the diagram in real-time
5. Download as PNG or SVG

REGENERATING DIAGRAMS
================================================================================

If you need to regenerate the diagrams after editing:

Option 1: Online (Easiest)
  1. Edit the .mmd file
  2. Go to https://mermaid.live/
  3. Paste the code

Option 2: Command Line (Requires mermaid-cli)
  npm install -g @mermaid-js/mermaid-cli
  mmdc -i architecture_enhanced.mmd -o output.png -b white

Option 3: Python Script (Already provided)
  python render_enhanced.py
  (Uses mermaid.ink API to render)

TIPS FOR POWERPOINT PRESENTATION
================================================================================

1. Use Enhanced Diagram for Main Architecture Slide
   - Shows complete system overview
   - Color-coded for easy understanding
   - Professional appearance

2. Use Simplified Diagram for Executive Summary
   - Less overwhelming for non-technical audiences
   - Focus on main flow

3. Use SVG Format When Possible
   - Better quality at any size
   - Can be edited in PowerPoint if needed

4. Add Animations
   - Reveal layers progressively
   - Highlight specific components
   - Show data flow step-by-step

5. Complement with Text
   - Use the ARCHITECTURE_OVERVIEW.txt for detailed notes
   - Add speaker notes from the overview document
   - Reference specific components during presentation


================================================================================
TROUBLESHOOTING
================================================================================

Issue: Diagram doesn't render properly
Solution: 
  - Check syntax at https://mermaid.live/
  - Ensure all subgraph/end pairs match
  - Verify node IDs are unique

Issue: Colors not showing
Solution:
  - Check classDef definitions
  - Ensure class assignments reference correct nodes

Issue: Layout looks wrong
Solution:
  - Try different graph types (TB, LR, etc.)
  - Adjust subgraph organization
  - Use direction hints within subgraphs

Issue: Text too small
Solution:
  - Edit fontSize in theme variables
  - Use <small> tags for subtitles
  - Export at higher resolution
